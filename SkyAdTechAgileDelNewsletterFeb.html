Top tips: Miro
If you are considered an owner of the board (you have a Miro license, have been given Edit access to the board and are in the team the board belongs to) then you can set timers and polls:
To set a timer, click the Clock icon – this will allow you to set the length of time, and optionally play some music. All contributors working on the board will see the timer count down. Click the Square with Dots icon to set up a poll – you will be able to set up the voting area and filter which objects people can vote on. You can also control the number of votes people can cast, and the length of time for them to vote – at the end you will see the results.

As with all these it is worth having a play before you use them in a session!
 
 
Top tips Jira:
This tip is for people using Advanced Agile Kanban Boards and noticing that items that are Done remain on their board (forever).

You can change this to be the same as the normal Kanban board (items stay in Done for 2 weeks before they are no longer visible).

On your board, click the cog next to the board name near the top:
Then adjust the JQL of the board by adding and (resolutiondate is EMPTY or resolutiondate >= -14d) to the end:
Learn more about JQL syntax here.
